Getting Started

Installing Compass
- gem install compass

- npm install -g yo
- npm install -g generator-angular
- npm install
- Bower Install
- Grunt
- Grunt server

Run either
- grunt test (test your app)
- grunt server (preview your app)
- grunt (build the application for deployment)
